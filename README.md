# Minecraft-CTF-Plugin
A Capture the Flag Plugin for Minecraft 1.15.2 Bukkit 

- Create teams with `\st <name>`
- Join a team then set your team's flag spawn point with `\sfs`
- Set a respawn timer for players `\rt <seconds>`

```
commands:
  setteam:
    aliases: st
  teams:
  setflagspawn:
    aliases: sfs
  cmdtoggle:
    aliases: cmtg
  respawntime:
    aliases: rt
```
